==Работа с файлами==

1. touch - создание пустых файлов или их изменение 

2. ls - просмотр содержимого директории 
3. pwd - показать полный путь текущей директории
4. rm - удаление файла или файлов 
5. mkdir - создание директории 
6. rkdir - удаление директории 
7. cp - копирование файлов и директорий
8. mv - перемещение файлов и директории

==Работа с текстом== 

Cat - просмотр содержимого файла 
tac - просмотр содержимого файла, начиная с последней строки 
head - вывод первых строк
tail - вывод последних строк 
more - просмотр содержимого файла "в одном экране"
less -  просмотр содержимого файла с прокруткой 
grep - поиск строк по образцу
sort - сортировка строк 
wc - подсчёт строк, строк и букв

# Типы менеджеров Пакетов
1) DPKG
2) APT
3) APT-GET
4) RPM
5) YUM
6) DNF

RPM (.rpm) напрямую не работает с зависимостями, приходится при установке делать все самому 

Установка пакета
```bash
rpm -ivh далее название пакета 
```
Удаление пакета
```bash
rpm -e telnet 
```

Обновление  пакета
```bash
rpm -Uvh telnet.rpm
```

Информация  пакета
```bash
rpm -q telnet.rpm
```

Проверка  пакета
```bash
rpm -V telnet.rpm
```
