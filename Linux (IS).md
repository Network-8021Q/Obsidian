# Разрешение файла

```bush
chmod <permissions> file изменение прав доступа к файлам и каталогам
```

```bush
 chmod u+rwx my-file полный доступ для owner
```

```bush
 chmod ugor+r-x my-file дать чтение для owner, group и other, убрать выполнение
```

```bush
 chmod o-rwx my-file убрать весь доступ для other
```

# ПРАКТИКА 


1) Какие разрешения установлены для `группы`(`group`) пользователей для той же директории `games`?
   Ответ:  Запусти: `ls -ld /home/max/games` и посмотри разрешения, установленные для группы.  
          `r-x` означает, что у группы есть разрешения на `чтение и выполнение`

2) Какие права установлены для `других`(`others`) пользователей в каталоге games?
    Ответ:  Запусти: `ls -ld /home/max/games` и посмотри разрешения, установленные для других (`others`).  
          `---` означает, что не установлено разрешений для других!

3) В каталоге games был создан новый файл с именем `residentevil`
   Он имеет полные разрешения, обнови файл, чтобы группа и другие пользователи имели только разрешения на чтение и выполнение
```bush
 chmod 755 /home/max/games/residentevil 
```

6) Теперь, добавь обратно разрешение на `запись` для `группы` и удали все разрешения для `других` для того же файла с названием `residentevil`
```bush
 chmod 770 /home/max/games/residentevil
```

7) Теперь измени владельца файла с именем `residentevil` на `service account` с названием `pollme`
```bush
 sudo chown pollme /home/max/games/residentevil
```

9) Мы создали еще один файл в каталоге игр. Измени владельца для всего каталога `games`, включая все файлы внутри, на служебную учетную запись `pollme`
   Попробуй сделать это с помощью единственной команды. Изучи рекурсивный флаг `-R` в команде `chown`
```bush
  sudo chown -R pollme /home/max/games
```
